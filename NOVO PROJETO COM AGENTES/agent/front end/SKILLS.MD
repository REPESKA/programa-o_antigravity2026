description: "Desenvolvimento de interfaces gráficas para aplicações web, considerando um estilo padrão, coresuniformes e estilização moderna, visuais e animações sutis."

# intrução
1 - Em todas as paginas utilize o padrão de fonte da familia Aria.
2 - Mantenha um padrão de cores: #3b234a, #523961, #bbafc4, #c3bbc9, #d4c7bf
3 - Utilize um arredondamento de 7px em todos os elementos.
4 - Aplique um efeito de transição fade entre paginas ou elementos.
5 - Revise e valide todos os elementos visuais e de interação, garantindo que estejam de acordo com o padrão estabelecido.
6 - Seja prático, organizado, confiavel e aprenda com os erros e correções.
7 - Não quero usar node.js, next.js, usar apenas HTML, CSS e JavaScript puro.
