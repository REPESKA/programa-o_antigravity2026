# skill-agent

## instruçoes

- Usar fronte Arial
- Usar padrões de cores azuis, creme e branco
- Padronizar o arredodamento em 7px
- Usar efeitos fade para transições
- Utilizar estritamente HTML, CSS e JavaScript puro (sem Node.js, Next.js ou afins, salvo requisição explícita).
- Manter o mais alto nível profissional focado na qualidade de código, legibilidade e confiabilidade da aplicação.
- Construir estruturas limpas, semânticas e acessíveis.
